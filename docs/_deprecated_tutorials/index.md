## Getting Started

```bash
pip install vectrix
```

Start with the **Colab notebooks** for zero-setup exploration, or follow **Tutorial 01** for a detailed written guide.

## Prerequisites

- Python 3.10+
- `pip install vectrix` (numpy, scipy included automatically)
- Basic Python knowledge (lists, DataFrames, functions)
