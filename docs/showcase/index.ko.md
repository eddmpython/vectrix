## 쇼케이스 설명

### 01 — 한국 경제 데이터 예측

FRED(미국 연방준비은행 경제 데이터)에서 제공하는 한국 경제 지표 예측:

- **원/달러 환율** — 1981년부터 월간 데이터, 12개월 예측
- **KOSPI 주가지수** — 월간 주가지수, 12개월 예측
- **소비자물가지수 (CPI)** — 인플레이션 추적, 12개월 예측
- **다중 지표 DNA 분석** — 4개 지표의 난이도와 특성 비교

모든 데이터는 `fred.stlouisfed.org`에서 자동 다운로드됩니다 — 회원가입 불필요.

### 02 — 한국 실제 데이터 회귀분석

한국 데이터셋을 활용한 회귀분석:

- **서울 자전거 대여량** — 8,760개 시간별 관측치, 기상 요인으로 대여 수 예측 (UCI 머신러닝 저장소)
- **한국 거시경제 회귀** — 환율 결정요인: 국채 수익률, 실업률, 주가지수, CPI (FRED)
- **자전거 데이터 → 시계열** — 동일 데이터셋으로 회귀분석과 시계열 예측 모두 수행
