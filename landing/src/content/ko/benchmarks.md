# 벤치마크

Vectrix의 M3 및 M4 Competition 벤치마크 결과입니다. 모든 결과는 공식 대회 데이터셋 전체를 대상으로 측정되었습니다.

## M4 Competition 결과

M4 Competition은 100,000개의 시계열로 구성된 세계 최대 예측 벤치마크입니다. 6개 빈도 그룹(Yearly, Quarterly, Monthly, Weekly, Daily, Hourly)에 걸쳐 다양한 도메인의 데이터를 포함합니다.

## 속도 벤치마크

Rust Turbo 모드 활성화 시의 성능 (Python 순수 구현 대비):

| 연산 | Python | Rust Turbo | 배속 |
|------|:------:|:----------:|:----:|
| AutoETS fit | 348ms | 32ms | 10.8x |
| AutoARIMA fit | 195ms | 35ms | 5.6x |
| Theta fit | 1.3ms | 0.16ms | 8.1x |
| DOT fit | 68ms | 2.8ms | 24x |
| AutoCES fit | 118ms | 9.6ms | 12x |
| FourTheta fit | 63ms | 5.6ms | 11x |
| ETS filter loop | 0.17ms | 0.003ms | 67x |
| ARIMA CSS objective | 0.19ms | 0.001ms | 157x |
| `forecast()` E2E | 295ms | 52ms | 5.6x |

> **참고:** 벤치마크는 단일 코어, Intel i7-13700H, Python 3.12 환경에서 측정되었습니다. Rust Turbo는 `pip install vectrix`로 설치 시 자동으로 포함됩니다 (사전 빌드된 wheel).

## 재현

벤치마크를 직접 재현할 수 있습니다:

```python
from vectrix import forecast, compare

result = forecast(data, steps=steps)
comparison = result.compare()
print(comparison)
```

M4 전체 데이터셋 벤치마크 스크립트는 저장소의 `src/vectrix/experiments/modelCreation/` 디렉토리에 포함되어 있습니다.
